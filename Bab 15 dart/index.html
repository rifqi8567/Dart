BAB 15 Asynchronous & Testing Future, Stream, dan Unit Testing

Kuasai konsep pemrograman Asynchronous (Future, async-await, dan Stream) untuk penanganan proses non-blocking. Serta pelajari cara merancang pengujian otomatis (Unit Testing) menggunakan package resmi Dart guna menjamin kestabilan dan keandalan kode Anda!

Tujuan Pembelajaran

Async & Event Loop Memahami konsep asynchronous dan cara kerja Event Loop single-threaded di Dart.
Future & Stream Mampu memproses data tunggal (Future) dan aliran data berkala (Stream) secara non-blocking.
async & await Menggunakan kata kunci async/await serta penanganan error via blok try-catch.
Unit Testing Mampu menulis test suite dengan package test untuk menjamin kebenaran logika program.
