BAB 9 Functions Dasar & Parameter

Pahami modularisasi kode menggunakan fungsi: cara pendeklarasian, pengembalian nilai (return value), parameter posisi opsional, keunggulan parameter bernama (named parameters) ala Flutter, dan nilai default!

Tujuan Pembelajaran

Deklarasi Fungsi Memahami struktur fungsi dasar, penentuan tipe return, dan fungsi void.
Positional Param Mampu menggunakan parameter berbasis posisi, baik yang bersifat wajib maupun opsional [].
Named Param Menguasai named parameter {} dengan required untuk meningkatkan kejelasan kode.
Default Value Mampu memberikan nilai default pada parameter opsional untuk efisiensi pemanggilan.
